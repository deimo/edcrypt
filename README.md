## edcrypt

#### 简介
Python语言下基于pycryptodome库的加解密算法Demo，可以作为个人代码wiki

**tip：如果你对与数据的加解密过程和算法还没有一个简单了解，您可以选择阅读一下本人的博文
[传送们]()**

#### 如何使用
 - 安装Python3

 - 安装pycryptodome
     ```
     pip install pycryptodome
     ```
 - 进行测试
     ```
     # 测试AES加解密过程
     python run aesdemo
     # 测试RSA加解密过程
     python run rsademo
     ```
